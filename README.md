# Job Application Tracker

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen)](https://job-application-tracker-amber.vercel.app/)


A React + TypeScript web application to track job applications, their statuses, and key details in one place.

The app focuses on clear state management, typed domain models, and a clean, responsive UI suitable for real-world usage.

## Features

- Add, edit, and delete job applications
- Filter applications by:
  - Status
  - Job type
  - Location type
- Color-coded status labels for quick visual scanning
- Responsive layout for desktop and mobile
- Data persisted in browser local storage
- Modal-based create and edit flows

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
